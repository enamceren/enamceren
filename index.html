XII TKJ 1 - Pada tahun 2023 semester ganjil, kami ditugaskan membuat web server di Debian Linux, service yang kami gunakan adalah Nginx.

Selain web server, ada DHCP server sebagai pemberi alamat IP secara otomatis kepada client.

FTP server sebagai media upload & download folder/file dari server ke client maupun dari client ke server.

Remote server menggunakan SSH fungsinya agar server bisa dikendalikan oleh client dalam jaringan lokal, sehingga client bisa mengkonfigurasi server melalui software seperti putty maupun melalui command prompt.

Yang terakhir DNS server yang berperan sebagai penerjemah nama Domain menjadi alamat IP, sehingga website bisa diakses menggunakan Domain.

Sesuai dengan intruksi guru pembimbing, setelah website sudah jadi untuk selanjutnya di upload ke hosting agar bisa dilihat secara online.
