Tutorial membuat web server menggunakan Nginx di Windows
Dipublikasikan pada 17 September 2023

M. Rifki Sumarna - Ini adalah konten blog pertama Namen. Pada kesempatan kali ini saya akan membagikan tutorial membuat web server di Windows menggunakan Nginx. Simak tutorialnya berikut ini.

Pertama yang kita butuhkan adalah nginx versi terbaru. Kunjungi situs https://nginx.org, kemudian klik Download.

Pilih yang Stable Version, kemudian klik nginx/Windows-1.24.0 pgp, maka paket nginx akan secara otomatis di download, tunggu proses download hingga selesai.

Buka File Explorer lalu temukan file nginx yang sudah di download tadi. Biasanya terletak di folder Downloads dengan nama 'nginx-1.24.0.zip'.

Ekstrak file nginx dengan cara klik kanan lalu akan muncul menu pilihan. Ada dua metode ekstrak, yang pertama bisa pilih Extract All... atau jika punya software tambahan misalnya winRAR maka kamu bisa ekstrak menggunakan opsi Extract to (nama file).

Setelah di ekstrak, buka folder nginx-1.24.0. Klik dua kali file nginx.exe untuk menjalankan service nginx.

Untuk melihat service nginx sudah berjalan atau belum yaitu melalui Task Manager, caranya klik kanan Logo Windows, lalu pilih Task Manager. Scroll ke bawah cari nginx.exe (32 bit). Jika sudah ketemu maka Web Service sudah berhasil kita buat.

Selanjutnya kita konfigurasi isi Website-nya, letak file website berada di folder html. Isi konten website bisa diisi mengenai data kelompok, proses membuat Web Server dan lain sebagainya.

Biasanya saya menggunakan template website dari situs https://free-css.com/free-css-templates. Di situs ini ada banyak pilihan website, ada versi free dan juga premium.

Kamu bebas pilih template yang mana saja, kamu bisa melihat tampilan website-nya sebelum kamu pakai dengan klik tombol LIVE DEMO. Apabila sudah cocok dengan pilihanmu, klik tombol DOWNLOAD.

Template website yang sudah kamu download harus di ekstrak terlebih dahulu karna format file-nya .zip. Ada dua pilihan untuk mengekstrak, bisa langsung Extract All... atau jika punya software tambahan misalnya winRAR maka kamu bisa ekstrak menggunakan opsi Extract to (nama file).

Pindahkan semua file template website menggunakan opsi Cut, melalui shortcut keyboard CTRL+A untuk memilih semua folder/file, lalu CTRL+X untuk memotong/cut berkas yang telah di blok.

Setelah itu masuk ke direktori tempat folder nginx berada. Selanjutnya CTRL+V untuk menempelkan/paste semua berkas yang telah di cut tadi ke folder html.

Jika kamu jago ngoding dan sudah punya konten website sendiri, maka pindahkan (cut) atau salin (copy) semua berkas website kamu ke direktori nginx.

File yang sudah di upload ke nginx bisa di ubah kodingannya menggunakan text editor seperti Notepad atau bisa juga di Visual Studio Code.

Yang terakhir kamu tinggal uji coba tes di browser, ketik localhost dan lihat hasilnya.

Sampai di sini berarti kita sudah berhasil membuat web server menggunakan Nginx di Windows.
